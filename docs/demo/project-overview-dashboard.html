Kiến trúc đã chọn
Headless Odoo v19 — Odoo là backend API server thuần túy
Quyết định kiến trúc
Không dùng website_sale, QWeb templates, hay Odoo native /shop. Odoo chỉ là backend. Frontend hoàn toàn tách biệt bằng Next.js. Kết nối qua REST API + JWT.
Tại sao không dùng Odoo Website native?
❌ QWeb templates khó tùy biến UX sâu
❌ Không tối ưu được performance frontend
❌ Khó tích hợp SSR/SSG cho SEO
❌ Phụ thuộc Odoo theme system
Stack OCA — "Không tự build lại bánh xe"
OCA FastAPI Expose Odoo models qua async FastAPI endpoints
ShopInvader Full eCommerce API: catalog, cart, checkout, orders, wishlist
JWT Auth fastapi_auth_partner_jwt — map token → res.partner
INHERIT ~38% endpoints dùng nguyên từ ShopInvader
EXTEND ~20% endpoints mở rộng thêm trường tùy biến
CUSTOM ~30% endpoints mới: Returns, P&L, Combo, Routing
Multi-Website / Multi-Company
1 Odoo Database
├── Website A: coolmate.me → CoolCash ON, CoolClub ON
└── Website B: brand-b.com → Standard Loyalty
